1. To use the sample code, you need to get your API keys by registering for an account at our website <http://www.mapmyindia.com/apis/>

2. Copy and paste the JavaScript API key in the sample code for interactive map development.

3. Copy and paste the REST API key in the sample code for calling the MapmyIndia Map RESTful web services (Geocoding, Reverse Geocoding, Routing, Distance, Map Image etc...)

4. The sample code is provided to help you understand the very basic functionality of MapmyIndia APIs. For any support, send us an email at apisupport@mapmyindia.com and we will get back to you as soon as possible.
